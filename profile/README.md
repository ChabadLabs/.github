# ChabadLabs

Open-source tools and AI-powered assistants for shluchim.

## GabayAI

A personal Shlichus AI assistant that runs on WhatsApp, powered by [nanoclaw](https://github.com/qwibitai/nanoclaw). GabayAI helps shluchim with scheduling, answering questions, managing groups, and more.
